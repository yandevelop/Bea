# Bea
  Bea adds small enhancements to the BeReal. app.

## Installation
  1. Add https://havoc.app to your package manager.
  2. Install Bea

## Compatibility
All iOS devices running iOS 14 or later.

## Build
  - Clone this repository using `git clone https://github.com/yandevelop/Bea`
  - `cd Bea`
  - Run `make package`
